---
title: What is HTML?
---

Whether you're building a static site, a blog, an e-commerce store or a complex web application, HTML is always at the heart.

But what does it mean to write HTML "well" or "badly"? How can we know if we are following best practices? In this class we will look at some guidelines and techniques that can help keep us on track, but we will almost always be coming back to one important concept:

### HTML is for content.
